# Callouts Maps

Coloca aquí las imágenes de callouts de cada mapa con el siguiente formato de nombre:

- `de_ancient.webp`
- `de_anubis.webp`
- `de_dust2.webp`
- `de_inferno.webp`
- `de_mirage.webp`
- `de_nuke.webp`
- `de_overpass.webp`

Las imágenes se mostrarán en el modal al hacer clic en el botón "Callouts Map" dentro de cada mapa.
